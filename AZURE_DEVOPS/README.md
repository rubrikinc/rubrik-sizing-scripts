# Azure DevOps Sizing Script

## Prerequisites

### Generate a Personal Access Token (PAT)
You'll need a PAT with Project Collection Administrator access to run the script. Here's a document on how to generate a PAT:
https://learn.microsoft.com/en-us/azure/devops/organizations/accounts/use-personal-access-tokens-to-authenticate?view=azure-devops&tabs=Windows

## Usage

### Prepare the Script
Copy and paste the script into a PowerShell window OR rename the attached text file to `GetAllRepos_latest.ps1`.

### Run the Script
Execute the PowerShell script.
